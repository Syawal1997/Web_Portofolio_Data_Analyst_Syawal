Dashboard real-time untuk memantau On-Time Delivery (OTD), rata-rata waktu tunggu, dan throughput gudang. Data diambil setiap 30 detik.

# Simulasi koneksi real-time & monitoring KPI
import pandas as pd
import psycopg2
import streamlit as st
from datetime import datetime

def fetch_realtime_metrics():
    conn = psycopg2.connect("dbname=logistics user=analyst")
    query = """
        SELECT 
            COUNT(CASE WHEN status = 'delivered' AND delivered_at <= eta THEN 1 END)::float 
            / NULLIF(COUNT(*),0) AS otd_rate,
            AVG(EXTRACT(epoch FROM (loaded_at - arrived_at))/60) AS avg_wait_minutes
        FROM shipments
        WHERE date_trunc('hour', created_at) = date_trunc('hour', now())
    """
    df = pd.read_sql(query, conn)
    conn.close()
    return df

# Streamlit Auto-refresh tiap 30 detik
st.set_page_config(page_title="Logistik Real-Time", layout="wide")
st.title("📊 Monitoring Operasional")
metrics = fetch_realtime_metrics()
st.metric("On-Time Delivery (Last Hour)", f"{metrics['otd_rate'][0]:.1%}")
st.metric("Rata-rata Waktu Tunggu", f"{metrics['avg_wait_minutes'][0]:.1f} menit")
Insight & Dampak Bisnis: Dashboard memungkinkan tim gudang mendeteksi lonjakan waktu tunggu >30 menit dalam 5 menit. Hasil: OTD meningkat 12% dalam 2 bulan.

🧹 Pembersihan Database Customer (12 Juta Record)

Python pandas FuzzyWuzzy SQL (Dedupe CTE) Open in Colab

Mengidentifikasi dan membersihkan 215.000+ duplikat pelanggan berdasarkan nama, email, dan nomor HP dengan fuzzy matching.

from fuzzywuzzy import fuzz, process
import pandas as pd

df = pd.read_csv('customers.csv')

# Deduplikasi berdasarkan nama dengan threshold 85%
def deduplicate_customers(df, threshold=85):
    unique_records = []
    seen_names = set()
    for idx, row in df.iterrows():
        name = row['customer_name']
        match = process.extractOne(name, seen_names, scorer=fuzz.token_sort_ratio)
        if not match or match[1] < threshold:
            unique_records.append(row)
            seen_names.add(name)
    return pd.DataFrame(unique_records)

cleaned_df = deduplicate_customers(df)
print(f"Before: {len(df):,} rows → After: {len(cleaned_df):,} rows")
# Output: Before: 12,410,000 → After: 12,194,832 (215k duplikat)
Insight: 40% duplikat berasal dari input manual CS. Rekomendasi: Implementasi validasi form & dropdown alamat otomatis. BI Impact: Menghemat biaya SMS blast hingga 18% per kuartal.

import pandas as pd

erp = pd.read_csv('erp_stock.csv')

merged = erp.merge(wms, on='sku', suffixes=('_erp', '_wms'))

merged['diff_all'] = merged[['qty_erp','qty_wms','qty_manual']].std(axis=1)
