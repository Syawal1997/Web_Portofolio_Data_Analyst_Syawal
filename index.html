🚚 Real-Time Logistics KPI Dashboard

Python Streamlit PostgreSQL Plotly Open in Colab

Real-time dashboard monitoring On-Time Delivery (OTD), average wait time, and warehouse throughput. Data refreshes every 30 seconds from API and internal database.

# Real-time KPI monitoring with Streamlit
import pandas as pd
import psycopg2
import streamlit as st

def fetch_realtime_metrics():
    conn = psycopg2.connect("dbname=logistics user=analyst")
    query = """
        SELECT 
            COUNT(CASE WHEN status = 'delivered' AND delivered_at <= eta THEN 1 END)::float 
            / NULLIF(COUNT(*),0) AS otd_rate,
            AVG(EXTRACT(epoch FROM (loaded_at - arrived_at))/60) AS avg_wait_minutes
        FROM shipments
        WHERE date_trunc('hour', created_at) = date_trunc('hour', now())
    """
    df = pd.read_sql(query, conn)
    conn.close()
    return df

st.metric("On-Time Delivery", f"{fetch_realtime_metrics()['otd_rate'][0]:.1%}")
st.metric("Avg Wait Time", f"{fetch_realtime_metrics()['avg_wait_minutes'][0]:.1f} min")
Business Impact: Detected wait time spikes >30 min within 5 minutes. Result: OTD improved by 12% in 2 months.

🧹 Customer Database Deduplication (12M Records)

Python FuzzyWuzzy Pandas SQL Open in Colab

Identified and cleaned 215,000+ duplicate customer records using fuzzy matching on name, email, and phone number.

from fuzzywuzzy import fuzz, process
import pandas as pd

df = pd.read_csv('customers.csv')

def deduplicate_customers(df, threshold=85):
    unique_records = []
    seen_names = set()
    for _, row in df.iterrows():
        name = row['customer_name']
        match = process.extractOne(name, seen_names, scorer=fuzz.token_sort_ratio)
        if not match or match[1] < threshold:
            unique_records.append(row)
            seen_names.add(name)
    return pd.DataFrame(unique_records)

cleaned = deduplicate_customers(df)
print(f"Before: {len(df):,} → After: {len(cleaned):,}")
# Output: Before: 12,410,000 → After: 12,194,832
Insight: 40% of duplicates came from manual CS input. Recommendation: Implement form validation & auto-address dropdown. Impact: SMS blast costs reduced by 18% per quarter.

🔄 Multi-Source Stock Reconciliation (3 Systems)

SQL Python Tableau QC Dashboard Open in Colab

Automated pipeline comparing daily stock from ERP SAP, WMS, and manual warehouse reports.

import pandas as pd

erp = pd.read_csv('erp_stock.csv')
wms = pd.read_csv('wms_stock.csv')
manual = pd.read_csv('manual_stock.csv')

merged = erp.merge(wms, on='sku', suffixes=('_erp', '_wms'))
merged = merged.merge(manual, on='sku')
merged['diff_all'] = merged[['qty_erp','qty_wms','qty_manual']].std(axis=1)

anomalies = merged[merged['diff_all'] > (merged['qty_erp'] * 0.05)]
print(f"SKU with anomalies: {len(anomalies)}")
Insight: 127 SKUs had >10% stock discrepancy, mainly due to delayed warehouse updates. Action: Auto-notifications to WhatsApp every 10 AM. Result: Stock accuracy improved from 87% → 98.5% in 3 months.
