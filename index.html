📉 Customer Churn Prediction & RFM Segmentation

Python Scikit-learn RFM SHAP Colab

📌 Apa: Model prediktif untuk mengidentifikasi pelanggan berisiko churn dan segmentasi RFM (Recency, Frequency, Monetary).

❓ Kenapa: Tingkat churn pelanggan mencapai 18% per kuartal, mengakibatkan penurunan pendapatan berulang. Tim marketing membutuhkan targeting yang lebih presisi.

🔧 Solusi: Analisis 500.000+ transaksi dengan RFM segmentation dan Random Forest untuk feature importance. Hasilnya diintegrasikan ke CRM untuk campaign otomatis.

# RFM Segmentation & Churn Prediction
import pandas as pd
from sklearn.ensemble import RandomForestClassifier

rfm = df.groupby('customer_id').agg({
    'date': lambda x: (pd.Timestamp.now() - x.max()).days,
    'invoice': 'count',
    'amount': 'sum'
}).rename(columns={'date':'recency','invoice':'frequency','amount':'monetary'})

rfm['churn'] = (rfm['recency'] > 90).astype(int)
model = RandomForestClassifier().fit(rfm[['recency','frequency','monetary']], rfm['churn'])
Hasil: Retention meningkat 9% dalam 2 bulan. ROI campaign reactivation mencapai 4.2x. Model diadopsi sebagai KPI churn bulanan oleh tim marketing.
